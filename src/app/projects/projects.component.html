Proyectos

Estos son algunos de mis proyectos

Sistema de administración de proyectos | |

Una solución escalable para un sistema enfocado en la gestión de proyectos a través de la autenticación de usuarios, asignación de roles y permisos. Puede interactuarse a través de una interfaz manejable para cualquier usuario.

Aplicación para la gestión de inventarios | |

Con un servidor que se conecta a dos bases de datos las cuales se ejecutan en paralelo en contenedores individuales de Docker, se logra la separación de responsabilidades en la autenticación de usuarios y los servicios centrales de la aplicación.

Sistema de administración y búsqueda de bienes raíces | | | |

El propósito de este proyecto es brindar a los clientes y agentes inmobiliarios una manera sencilla y objetiva para navegar por diferentes ofertas publicadas por los propietarios de cualquier tipo de inmueble.
